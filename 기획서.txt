1 사용자 권한
◆ Teacher (선생님)

본인 시간표 전체 열람 가능
출석 상태 확인 가능
일정 시간 변경은 “수정 요청”만 가능 (관리자 승인 후 반영)
공지사항 열람 및 확인 버튼 클릭 가능

◆ Admin (원장/부원장)

전체 시간표 열람 가능
일정 생성/수정/삭제 가능
선생님/학생 추가·수정·비활성화 가능
출석 및 보강 최종 관리
공지 작성/수정/삭제 및 확인 현황 확인 가능
※ 전체 시간표는 Admin만 열람 가능

2 기본 화면

앱 실행 시 월간 달력(한 달 보기)가 기본 화면
날짜 클릭 → 수업 상세 확인
선생님 또는 학생 이름 클릭 시 출석 상태 확인 가능

3 출석 관리
출석 상태는 3가지로만 운영:

출석
결석
보강

4 보강 관리

결석 시 보강 여부 확인 가능
보강 예정일 입력 가능
보강 완료 처리 가능
결석과 보강은 서로 연결되어 추적 가능
예:
3/5 결석 → 3/20 보강 예정 → 보강 완료

5 일정(수업시간) 관리

일정 생성: Admin만 가능
수정 가능 항목: 날짜, 시간, 담당교사
Teacher는 수정 요청만 가능
변경 이력 저장 필수 (누가/언제/어떻게 변경했는지 기록)

6 선생님/학생 관리
Admin만 가능:

선생님 추가/수정/비활성화
학생 추가/수정/비활성화

7 공지사항 기능

Admin이 공지 작성
Admin이 공지별 “확인” 클릭
Teacher가 열람 후 “확인” 클릭
Admin은 공지별 확인 여부 확인 가능
중요 공지는 상단 고정 또는 팝업 표시

☑ 필수 기능 체크

로그인 및 권한 구분
월간 달력 기본 화면
내 시간표 보기
전체 시간표(Admin 전용)
출석 3가지 상태 관리
결석-보강 연결 구조
일정 수정 요청 + 변경 이력
학생/선생님 관리
공지 확인 기능
